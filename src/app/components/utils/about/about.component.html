About XSD2XML

XSD2XML allows you to load or enter a XML Schema Defintion (XSD) which it uses to generate a dynamic form to allow you to build a valid XML document from the definiotn

It differs from similar tools by putting you completely in the driving seat by allowing you to quickly add or delete optional elements. The cardinality of elements is enforced and only the defined number of elements can be entered. Where the XSD defines a Choice, the choice is presented in a familiar dropdown selection box. Required element attributes are pre selected and optional attributes are easily enabled.
